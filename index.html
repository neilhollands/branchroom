To open a shared project

  1. 1

    Download the project media files.

  2. 2

    Choose Open project, then open a .branchroom.json file or paste JSON from an email.

  3. 3

    Choose Find missing files and select all the media files from the shared folder.

Keep media filenames unchanged. Branchroom uses them to reconnect the project.
